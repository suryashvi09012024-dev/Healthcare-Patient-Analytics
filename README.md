# 🏥 Healthcare Patient Analytics Dashboard – Transforming Healthcare Data into Actionable Insights

An end-to-end Healthcare Analytics project that analyzes patient demographics, treatment costs, hospital visits, recovery outcomes, and readmission risks using Python, Excel and Power BI.

This project transforms raw healthcare data into meaningful insights through data cleaning, exploratory data analysis (EDA), SQL analysis, and interactive Power BI dashboards to support data-driven healthcare decision-making.

# 📌 Project Overview

Healthcare organizations generate large amounts of patient data every day. This project analyzes healthcare records to identify treatment trends, patient demographics, hospital performance, and readmission risks.

# 💡 Key Insights

Most patients belong to the Adult and Middle Age groups.

General Medicine receives the highest number of patient visits.

Treatment costs vary significantly across departments.

Longer hospital stays are associated with higher treatment costs.

High recovery scores generally correspond to lower readmission risk.

Certain regions experience higher patient volumes, indicating increased healthcare demand.

Readmission risk is higher among patients with extended hospital stays.
